🔍 การดูข้อมูล
ข้อมูลซิงค์จาก Google Sheets อัตโนมัติเมื่อเปิดหน้า กดปุ่ม ซิงค์ เพื่อดึงข้อมูลล่าสุด
🔎 การค้นหาและกรอง

🔐 Admin Mode
กดปุ่ม Admin Login แล้วใส่รหัสผ่าน เพื่อเปิดสิทธิ์เพิ่ม / แก้ไข / ลบเอกสาร และ Export ข้อมูล
⚠ สีค้างวัน
สีเขียว < 7 วัน (ปกติ)   สีเหลือง ≥ 7 วัน (เริ่มล่าช้า)   สีแดง > 14 วัน (ล่าช้า)
📊 สถานะเอกสาร
Under Review — กำลังพิจารณา Approved — อนุมัติแล้ว สามารถนำไปใช้งานได้ Approve by Comments — อนุมัติตามบันทึก สามารถนำไปใช้งานได้แต่ต้องแก้ไขตามบันทึก Revise & Resubmit — ต้องแก้ไขและส่งกลับมาใหม่ ไม่สามารถนำไปใช้งานได้ Rejected — ไม่อนุมัติ ไม่ให้ใช้งาน Cancelled — ยกเลิกแล้ว
📦 Backup / Restore
ใช้เมนู Backup ใน Admin Bar เพื่อสำรองข้อมูลเป็นไฟล์ JSON และ Restore เมื่อต้องการนำเข้าคืน
